cornerstoneTools Examples

These examples are meant to demonstrate various aspects of the cornerstoneTools library in an interactive way. They are written such that they can be opened directly in your web browser from github or your local repository. Use the "view source" capabilities built into your browser to see how each example works.

  • AllImageTools - see all the image tools working together in one example
  • MobileTouch - example of using touch events
  • All Touch Tools - showing off all the touch-capable tools
  • Pixel Probe Tool - example of using the pixel probe tool
  • Drag Probe Tool - example of using the drag probe tool, which disappears on mouseUp
  • Stack Scroll Tool - example of using the stack scroll tool
  • Stack Loading Progress - example of monitoring image loading progress within a stack
  • Pan/Zoom Synchronizer - example of synchronizing the pan and zoom between multiple images
  • Reference Lines Tool - example of displaying reference lines
  • Stack Image Position Synchronizer Tool - example of stack synchronization tool using the image position
  • Stack Image Position Offset Synchronizer Tool - example of stack synchronization tool to maintain the distance in patient space between two stacks
  • Stack Image Index Synchronizer Tool - example of stack synchronization tool using image index
  • Stack Scroll Synchronizer Tool - example of stack synchronization tool using scrolling
  • wwwcSynchronizer - example of synchronizing ww/wc and invert
  • highlight - rectangular highlight tool
  • timeSeries - time series tool
  • Tool Management - Clearing unwanted tool data
  • Save Image - example of using the saveAs function to download an element
  • Window By Region - example of using the window width/level by region tool
  • Rotate viewport - example of using the mouse / touch rotate tool
  • Image Annotation - image annotation with text and arrows
  • Magnifying glass - example of using the magnification tool to examine an image
  • Freehand ROI drawing - example of using the freehand tool to draw arbitrary points
  • WWWC Tool - example of using the WWWC tool
  • 3D Crosshairs Tool - example of a 3D crosshair tool
  • Image Orientation Markers - example of direction metadata to change HTML orientation markers
  • On-Canvas Orientation Markers - example of the orientationMarkers tool for drawing orientation markers inside the canvas
  • Text Marker Tool - example of the text marker (i.e. vertebrae marker) tool
  • Keyboard Stack Scrolling - example of using keyboard input to scroll through a stack
  • Angle Tool - example of the simple angle tool
  • Load Handler Functions - example of setting and using load handler callbacks
  • Application State Save / Restore - example of easily saving and restoring viewport and tool data for a set of elements
  • Seed Points - image annotation for seed points
  • Image Statistics - display debugging information on rendering performance
  • Segmentation Brush - example of a simple paint brush
